Operational Workflow

1. TRIGGER

Cradle status change detected. Metadata captured via Browser Extension.

2. ACQUIRE

Multi-source fetching: Cloud CDN (Binary Blobs) or Network Paths (Lucid/SMB).

3. PROCESS

Media Engine decodes streams and isolates audio stems via AI separation.

4. ANALYZE

Signal processing: Visual SSIM, Audio LUFS, and linguistic transcription.

5. VERDICT

Agent 2 applies logic guardrails and reports final justification to Cradle.

Agent 1: The Controller

Control Loop: Periodic monitoring of Cradle DOM every 120s.

Acquisition Strategy:

  • Cloud CDN (Direct Blobs): High-speed binary streams directly from project URLs.
  • Network Shares: Automated path mapping for Lucid Link and local NAS volumes.
  • Relevance Logic: Precise file identification via Cradle ID, Template ID, and specific workflow steps.

Agent 1: Components

Role: Real-time DOM scanning and metadata harvesting directly from Cradle.

NVC Processing Engine

FFmpeg Integration: Native decoding with multi-platform hardware support.

Hardware Abstraction:

  • macOS: Native VideoToolbox acceleration.
  • Linux/Server: NVDEC via CUDA (Requires NVIDIA optimized code).
  • Other: Intel QSV or Software decoding (libx264).

Video: SSIM Algorithm

Structural Similarity Index (SSIM)

Mathematical model evaluating degradation across three human visual perception layers:

  • Luminance: Mean pixel intensity.
  • Contrast: Structural variance and standard deviation.
  • Structure: Correlation and covariance between shifted pixels.

Video: MSE & Heatmaps

Mean Squared Error

Global variance tracking. Essential for detecting color-space errors and bitrate drops.

In practice: Catching massive encoding failures that SSIM might classify as structural matches.

Visual Diff Masks

Generates Heatmaps highlighting coordinates where files differ, enabling rapid manual verification.

Audio: AI Source Separation

Demucs v4 (Hybrid Transformer)

AI model used to isolate the voiceover (VO) stem from complex background music.

  • Vocal Isolation: Prevents music interference during linguistic analysis.
  • Clean Input: Dramatically reduces Whisper hallucinations in action-heavy content.

Audio: EBU R128 Compliance

Loudness Normalization Standards

Standardized volume measurement to ensure broadcast and digital audio consistency.

  • Absolute Integrated Loudness: Measures integrated volume across the file (Target: -23 LUFS).
  • True Peak (dBTP): Measures signal peaks to prevent digital clipping.

STT: Faster-Whisper Engine

CTranslate2 Re-implementation

High-performance Whisper backend using CTranslate2 for up to 4x faster inference.

  • CTranslate2: Specialized inference engine utilizing custom kernels for Transformers.
  • Int8 Quantization: Halves VRAM usage (12GB -> 3GB) with near-zero accuracy loss.

Linguistic: AI Semantic Analysis

The system uses LLM-driven Semantic Analysis (Agent 2) to compare transcriptions.

  • Contextual Match: AI understands if a word variation is a phonetic artifact or an error.
  • Locale Check: Confirms spoken language correctly reflects the target market.

Agent 2: The Logic Brain

Processes multidimensional signal data to generate a final project verdict.

Quantitative Metrics

Hard math comparison: SSIM, MSE, LUFS, WER.

Qualitative Reasoning

Synthesizing data points into a readable explanation.

Glossary

SSIM: Structural Similarity Index (Perceptual image quality).

MSE: Mean Squared Error (Pixel variance).

LUFS: Loudness Units Full Scale (Audio volume standard).

WER: Word Error Rate (STT system performance).

AI Reasoning: Ollama & Llama 3

Agent 2 uses Llama 3 (running locally via Ollama) to generate reasoning.

  • Data Privacy: GDPR compliant; all inference stays within the local network.
  • Context Injection: Metrics are formatted into a prompt for descriptive reporting.

Safety: Deterministic Thresholds

Hard-coded Guardrail Thresholds prevent AI hallucinations.

  • FORCE REJECT if SSIM < 0.95 (Critical visual discrepancy).
  • FORCE REJECT if Audio Sync Offset > 2 frames (Noticeable desync).
  • FORCE REVIEW if STT Confidence < 0.7 (Unreliable transcription).
  • FORCE REVIEW if LUFS Difference > 1.5 (Significant volume gap).

Optimization & RAM

Operational stability strategies for resource-constrained hardware:

  • Sequential Lifecycle: Modules run serially to minimize peak memory load.
  • VRAM Flushing: Explicit torch.cuda.empty_cache() after AI stages.
